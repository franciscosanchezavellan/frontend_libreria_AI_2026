# frontend_libreria_AI_2026
frontend_libreria_AI proyecto final de electivas 2026

SISTEMA DE FACTURACIÓN LIBRERÍA EL ESTUDIANTE

Cómo usar:
1. Abrir index.html en el navegador.
2. Iniciar sesión con:
   - admin / 1234
   - empleado / 1234

Funcionalidad clave:
- El administrador puede crear, modificar y eliminar catálogos.
- El empleado puede consultar y registrar ventas.
- Las compras aumentan el stock del producto.
- Las ventas disminuyen el stock del producto.
- El sistema valida stock insuficiente antes de vender.
- El inventario muestra existencias y movimientos de entrada/salida.
- Los datos se guardan en localStorage del navegador.

Tecnologías:
- HTML5
- CSS3
- JavaScript
- LocalStorage
